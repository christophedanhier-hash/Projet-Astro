Bonjour ! Voici le contexte complet de mon projet pour cette session. Merci de le prendre en compte pour toutes tes réponses.

1. RÈGLES DE BASE :
   - Langue : Français uniquement.
   - Rôle : Expert en développement Web (Three.js/Canvas) et Mobile (Android/Capacitor).
   - Environnement : Je travaille sur un Chromebook (Linux/Debian).

2. LE PROJET : "Projet Astro"
   C'est une application de planétarium interactif et d'observation astronomique.
   - Stack technique : HTML5, CSS3, JavaScript (Vanilla + Modules), Three.js (pour la 3D).
   - Architecture : 
     - Dossier `www/` : Contient tout le code source web.
     - Dossier `android/` : Projet natif généré via Capacitor.
   - Objectif : Fournir une simulation précise (formules de Kepler) et visuelle du système solaire pour l'observation amateur.

3. LES MODULES CLÉS :
   - Simulateur 2D (`Simulateur_2D.html`) : Vue "du dessus" avec orbites, astéroïdes et calculs de visibilité locale.
   - Simulateur 3D (`Simulateur_3D.html`) : Vue immersive Three.js avec textures, fond étoilé et positionnement réaliste.
   - Module Lunaire (`Simulateur_Lune_3D.html`) : Focus sur les phases de la Lune, la libration et les syzygies.
   - Simulateur Terre (`Simulateur_Terre.html`) : Visualisation des saisons, de l'inclinaison axiale et de l'éclairement solaire.
   - Guide & Observations : Pages d'aide et carnet de notes.

4. ÉTAT ACTUEL :
   Le projet est en version 2.0 (Mise à jour majeure : Ajout du module Terre).
   - **Précision** : Les orbites sont calculées via l'équation de Kepler et les dates sont gérées en UTC.
   - **Interface (UI)** : Un calendrier personnalisé en HTML/CSS a été implémenté pour garantir la compatibilité sur toutes les plateformes. Un bouton "Aujourd'hui" a été ajouté.
   - **Compatibilité Mobile** : Le clic sur les planètes en 3D a été corrigé pour fonctionner sur les écrans tactiles (Android).
   - **Module Terre (Finalisé)** : Visualisation interactive des saisons, contrôle de l'inclinaison (Tilt), géolocalisation, texture jour/nuit, et affichage des dates de solstices/équinoxes.
   - **Accueil & Guide** : Nouvelle mise en page en grille pour l'accueil et mise à jour complète du guide utilisateur.

5. PROCHAINE ÉTAPE :
   Poursuivre le développement et l'optimisation sur l'ensemble des 4 simulateurs (2D, 3D, Lune, Terre) pour enrichir l'expérience utilisateur.
